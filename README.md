## FCIS Activities
## Flutter application to browse faculty of computer science ASU student activities

# Widgets used
- FutureBuilder
- ListView
- Drawer
- CustomScrollView
- SliverAppBar

# Activity Data
Data is fetched using http request to a JS API with the student activity data
